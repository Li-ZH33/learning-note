# 📖 笔记使用方法

# 所有笔记均为Markdown格式，推荐复制到Obsidian中阅读使用，适配性最佳：

# 1\. 进入仓库对应笔记文件夹，找到需要的Markdown文件（后缀为.md）；

# 2\. 点击文件右上角「Raw」，复制全部文本内容；

# 3\. 打开Obsidian，新建笔记，将复制的内容粘贴进去，即可正常阅读、编辑；

# 4\. Obsidian的双链、标签功能可更好地串联知识点，建议搭配Obsidian的笔记管理功能使用，提升学习效率。

# 注：直接在GitHub上阅读也可，但部分格式（如双链、特殊排版）可能无法正常显示，推荐优先使用Obsidian。
